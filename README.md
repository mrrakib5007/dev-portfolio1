# Developer Portfolio

A clean and modern **Developer Portfolio** built using **HTML5** and **CSS3**. This project showcases a personal portfolio website with sections for introduction, about, skills, resume, portfolio gallery, and contact information.

> **Note:** This is a desktop-only project and is not yet responsive.
